# LeoSync

Keep folders in sync across all your devices. Any file change on one machine automatically appears on the others.

## Install

### macOS / Linux

```bash
curl -fsSL https://raw.githubusercontent.com/LeoManrique/LeoSync/master/scripts/install.sh | bash
```

### Windows

Download the latest `.zip` from [Releases](https://github.com/LeoManrique/LeoSync/releases).

## Usage

1. Launch LeoSync
2. Sign in with your Google account
3. Add folders to sync
4. Repeat on your other devices — files sync automatically

## Self-hosted Relay

LeoSync requires a relay server to coordinate sync. You can self-host one using Docker — see the release notes for deployment details.
